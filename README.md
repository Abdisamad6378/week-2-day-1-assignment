# Week 2 Day 1 Assignment: JavaScript Fundamentals Drills

## Overview
This assignment covers JavaScript fundamentals including variables, data types, functions, conditionals, arrays, and objects through practical coding challenges.

## Tasks Completed

### Task 1: Five Classic Functions (20 points)
- **fizzBuzz(n)** - Prints numbers 1 to n with FizzBuzz logic
- **reverseString(str)** - Reverses a string
- **isPalindrome(str)** - Checks if a string is a palindrome (case-insensitive)
- **findLargest(arr)** - Finds the largest number in an array
- **countVowels(str)** - Counts vowels in a string

### Task 2: Kenyan County Object (25 points)
- Created county objects with properties (name, capital, population, area, borders)
- **displayCounty()** - Formats county information
- **formatPopulation()** - Formats numbers with commas
- **bordersString()** - Creates formatted border list with "and"

### Task 3: Matatu Route System (25 points)
- Created array of 10 matatu routes with name, fare, and stops
- **cheapestRoute()** - Finds the cheapest route
- **routesThroughStop()** - Finds all routes passing through a given stop
- **journeyFare()** - Calculates total fare for multiple routes
